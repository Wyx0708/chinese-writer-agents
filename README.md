# 中文创作协作引擎

> 基于多 Agent 长链推理的中文小说创作系统

## 项目简介

本项目构建了一个三 Agent 串联协作的中文小说创作引擎，解决单次 LLM 调用无法同时
处理"结构规划 → 散文生成 → 质量审校"三个认知层次迥异任务的核心痛点。

## Agent 架构
| Agent | 职责 | 输出 |
|-------|------|------|
| 策划 Agent | 解构需求，生成结构化大纲 | JSON（人物/场景/叙事结构/关键细节） |
| 写作 Agent | 依据大纲展开叙事 | 600-900 字中文正文 |
| 审校 Agent | 跨三层上下文一致性验证 | 五维评分 + 修改建议 |

## 核心逻辑流

- **长链推理**：每个 Agent 的输出作为下一个 Agent 的输入，形成完整推理链
- **多 Agent 协作**：三个 Agent 分工明确，各司其职
- **流式输出**：实时可视化每个 Agent 的生成过程

## 技术栈

- Frontend: React
- AI: Claude API (claude-sonnet-4)
- 架构: Multi-Agent Chain
